## A base construção Web

```js
A base dos site da web se baseiam nas linguagens Html , Css e linguagens de programação.
```

# Html 

```js
Basicamente o html é uma linguagem de marcação e hyper texto .

O html vai dar a estrutura para seu site.

-Hypertexto->Links que conectam as paginas

-Marcação -> Detalhamento através de TAGs

Exemplos : 

<html>,<body><head>...etc

```

# Css 

```js 

visual : Define a aparência dos elementos como cores,fontes tamanhos e estilos de bordas .

Organiza o layout da página ,controlando como os elementos são dispostos e como se comportam em diferentes tamanhos de tela

Responsividade : Permite que as páginas se adaptem a diferentes dispositivos e tamanhos de tela , como desktops , tablets e smartphones.

Manutenção e Consistência : Facilita a manutenção do estilo através da separação da apresentação e do conteúdo , permitindo uma estilização consistente e reutilizável em toda a aplicação.

```

# Java Script 

```js 

Linguagem de Programação

Basicamente é oque vai fazer a parada funcionar

Segue uma estrutura (sintaxe)  própria.

JavaScript é o que transforma uma página que nao funciona  em uma que funciona: cliques, animações, feedback, estados que mudam sem recarregar a pagina 

Manipulação do DOM(Document Object Model):O DOM  é a ponte entre JavaScript é o HTML real que o navegador renderiza. Manipular o DOM é, na prática, alterar a estrutura,conteúdo e o estado visual da aplicação em tempo real 

comnunicação assíncrona permite buscar e enviar dados sem travar a interface, usando fetch, async; await e APIs externa 
```
