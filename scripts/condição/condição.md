## if/else

```js 

O if e else  são estrututas condicionais que permite executar diversos blocos de códigos relacionada a condinção ou expressão
Exemplo :
 
let ocupado = 1

if (ocupado ==1){
    console.log("Ocupado")
}

else {
    console.log("Fazendo nada")
}

```

# if / Else if / Else 

```js 
Além disso, também é possível ultilizar múltiplas estruturas if para verificar  várias condições consecutivas, ou usar a estrutura else if para testar uma nova condição quando a primeira ou anteriores nao forem cumpridas

Exemplo :

let ocupado = 2

if (ocupado ==1){
    console.log("Ocupado")
}

else if (ocupado ==2) {
console.log("Compromisso")
}

else {
    console.log("Fazendo nada")
}

```
